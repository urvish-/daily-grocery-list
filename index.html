📲 Add to Home Screen

⚠️ Chrome on iPhone cannot install apps.
Apple only allows Add to Home Screen from Safari.

Follow these steps:

  1. Tap Copy link for Safari below
  2. Open the Safari app (blue compass icon)
  3. Paste the link in Safari's address bar and go
  4. In Safari, tap Install (top right) and follow steps

Install on iPhone / iPad using Safari:

  1. Tap the Share button ⬆️ at the bottom of Safari
  2. Scroll down → tap Add to Home Screen ➕
  3. Tap Add — app icon appears on your home screen

Install on Android (Chrome):

  1. Tap browser menu ⋮
  2. Tap Install app or Add to Home screen
  3. Confirm — icon appears on home screen

Install on Windows / Mac (Chrome or Edge):

  1. Click the install icon in the address bar, or
  2. Browser menu → Install Daily Grocery List
